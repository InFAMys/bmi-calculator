# BMI Calculator

Calculate Your Body Mass Index

## Prerequisites
* React JS (create-react-app)
```bash
npm install create-react-app -g
```

## Installation

First Run :

```bash
npm install
```

## Usage
To start the application run :

```bash
npm start
```
